# @editsy/editor

The editsy editing UI (Vite + React). You don't install this directly:
`@editsy/cli` and `@editsy/next` depend on it and serve its built `dist/`,
locally and from your deployed site respectively.

Start at [editsy](https://npmjs.com/package/editsy) or
[github.com/editsy/editsy](https://github.com/editsy/editsy). MIT.
